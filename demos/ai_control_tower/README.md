# AI Workflow Control Tower - TechGroei

A centralized command-and-control dashboard for managing multiple AI agents and human-in-the-loop (HITL) workflows.

## Features
- **Centralized Metrics**: Real-time tracking of health, cost, and latency.
- **Approval Queue**: Review, edit, and approve AI outputs before they reach clients.
- **Confidence Scoring**: Visual indicators of AI certainty to prioritize human review.
- **Activity Feed**: Live log of all background agent operations.
- **Audit Ready**: Full record of all system and human decisions.

## Tech Stack
- **UI**: HTML5 / CSS3 (Grid Layout, Glassmorphism, Dark UI)
- **Logic**: Vanilla JavaScript (Event-driven state management)
- **Icons**: Emoji Glyphs
- **Typography**: Outfit

## How to use
Open `index.html`. You will see a dashboard with a list of tasks requiring approval. Click any card to open the **Review Sidebar**, where you can approve or reject the AI-generated task. Monitor the "Live Activity Feed" on the right to see background agent tasks.
