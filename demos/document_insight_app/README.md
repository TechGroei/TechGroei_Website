# Document-to-Insight Workflow - TechGroei

A professional document intelligence tool that transforms unstructured data (PDFs, CVs, Contracts) into actionable business insights.

## Features
- **Intelligent Classification**: Automatically detects document type (Contract, Invoice, CV).
- **Entity Extraction**: Pulls critical metadata like dates, values, and parties.
- **Risk Detection**: Flags high-risk clauses and financial anomalies.
- **Actionable Roadmap**: Generates next steps based on document findings.
- **JSON Export**: Provides machine-readable data for ERP/CRM systems.

## Tech Stack
- **UI**: HTML5 / CSS3 (Glassmorphism & Flexbox)
- **Logic**: Vanilla JavaScript
- **Icons**: Emoji Glyphs
- **Typography**: Outfit

## How to use
Open `index.html` and click on one of the "Demo Document" chips to see the automated extraction and analysis in action.
