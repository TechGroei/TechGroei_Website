# AI Readiness Tool - TechGroei

A professional, technical readiness assessment tool designed to help companies evaluate their AI adoption potential.

## Overview

The AI Readiness Tool transforms vague AI ambitions into concrete, actionable implementation steps. It evaluates proposed use cases across four critical dimensions:
- **Data Quality & Accessibility**: Identifies if the data is structured, where it resides, and if it's usable for LLMs.
- **Architecture & Technical Fit**: Recommends hosting strategies (Cloud vs. On-Prem) and evaluates integration risks.
- **Governance & Compliance**: Highlights potential EU AI Act risks, PII concerns, and human-in-the-loop requirements.
- **Operational Risk**: Gauges the impact of AI hallucinations and failure modes.

## Features

- **Interactive Assessment Form**: Structured input gathering for use cases and technical constraints.
- **Dynamic Scoring Engine**: A rule-based algorithm that calculates a readiness score (0-100%).
- **Instant Report Generation**: Produces a structured Markdown report including:
  - Executive Summary
  - Risk Analysis
  - Quick Wins
  - 30-Day implementation Roadmap
- **Export Capability**: Download the generated assessment as a Markdown file for internal sharing.

## Tech Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Glassmorphism, Dark Mode, CSS Transitions/Animations)
- **Logic**: Vanilla JavaScript
- **Typography**: Outfit (Google Fonts)

## How to Run

1. Open `index.html` in any modern web browser.
2. Fill out the assessment questionnaire.
3. Click "Generate Assessment Report" to view your results.
4. Use the "Download Report" button to save your assessment.
